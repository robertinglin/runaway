Runaway

We have phones, tablets, phablets, watches, swatches, refrigerators, tvs, cars and even toasters connected to the internet thes days. The world doesn't just have 3 devices and we shouldn't be developing like it does!

Runaway is a different take on responsive design. Instead of lumping each device into a generic category, it lets them shine all on their own.

Boiled down Runaway is a set of wrapper, and helper mixins around a SASS mixin called spread. Spread, created by Doug Avery, allows developers to input value ranges for properties and automatically turns them into a sequential set of media query breakpoints. Runaway adds structure to spread with a preconfigured set of functions that help translate the idea to the more known concept of mobile, tablet, and desktop breakpoints.

There are 4 different types of mixins, r ( reveal ),s ( spread ),i ( inbetween ) and min-max.
Reveal helpers are single property sets for the configured base breakpoints. If you only want a property on the small breakpoint range you would use the r-small mixin. 
body {
  /* The background color will change depending on the viewport */
  @include r-small( background-color, #400 );
  @include r-med( background-color, #040 );
  @include r-large( background-color, #004 );
}
Spread helpers will spread the property values within the selected viewport. 
body {
  @include s-small( font-size, 30px, 12px );
  @include s-med( font-size, 1em, 2em );
  @include s-large( font-size, 10%, 30% );
  /* Max works with any size above the predetermined maximum */
  @include s-max( font-size, 30%, 50% ); 
}
Inbetween helpers act like spread helpers except they spread values inbetween multiple viewports. 
.floatingButton {
  @include i-small-med( left, 0px, 40px );
  /
  @include i-small-large( left, 40px, 20px, out-quad );
  @include i-small-max( left, 0px, 40px );

  @include i-med-large( left, 0px, 40px );
  @include i-med-max( left, 0px, 40px );

  @include i-large-max( left, 0px, 40px );
}
Min and max helpers are similar to r helpers in that they are basic media queries based off of the configuration. 
.randomBlockElement {
  @include min-small( min-height, 300px );
  @include max-small( min-height, 300px );

  @include min-med( min-height, 300px );
  @include max-med( min-height, 300px );

  @include min-large( min-height, 300px );
  @include max-large( min-height, 300px );

  @include min-max( min-height, 300px );
  @include max-max( min-height, 300px );
}
